# stone-lang
A testing project for creation of computer language

## Stone语法

### 数据类型
* 数值
* 字符串

### 表达式
#### 赋值表达式
```
sum = 0
```
#### 运算表达式
```
sum = sum + 1
```
#### 条件表达式
```
i < 10
```

### 语句

#### 分支语句
```
if i % 2 == 0 {
    even = even + 1
} else {
    odd = odd + 1
}
```

#### 循环语句
```
while i < 10 {
    i = i + 1
}
```
